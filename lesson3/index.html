Site purpose

The purpose of this site is meant to community website for Preston, Soda Springs, and Fish Haven. The website will give the user an accurate weather reading in their area. The user will be able to know if it is sunny, cloudy, raining, thunderstorms, and etc. The website will display the temperature, barometic pressure, air quality, and weather conditions. This website will also provide information on the before named cities attractions, shopping, eateries, schooling, and community events.
